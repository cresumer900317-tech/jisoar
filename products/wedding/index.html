Project · Photo Cloud

Wedding Memory Cloud

Guests upload photos & videos via QR.

Overview

하객이 부채에 인쇄된 QR을 찍으면 로그인 없이 사진·영상을 올리고, 신랑·신부는 한곳에서 모아 보고 ZIP으로 받는 결혼식 갤러리입니다.

Why I built it

결혼식의 가장 좋은 순간들은 정작 하객들의 폰 속에 흩어집니다. 그걸 한곳에 모으고 싶었어요. 핵심은 업로드 장벽을 0으로 — 앱 설치도, 로그인도, 이름 입력도 없이 QR 한 번이면 끝나게.

What it does

  • QR 스캔 → 로그인 없이 즉시 업로드
  • 사진 + 동영상 지원, 클라이언트 압축 · 병렬 업로드
  • 신랑·신부용 실시간 갤러리, 식장 스크린용 풀스크린 슬라이드쇼
  • 전체 ZIP 다운로드, 관리자 사진 회수
